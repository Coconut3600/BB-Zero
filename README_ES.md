## BB‑Zero — Resumen del Proyecto

BB‑Zero es un motor estilo AlphaZero para las damas clásicas españolas, basado en PUCT, Monte Carlo Tree Search, ruido Dirichlet, temperatura y aprendizaje por refuerzo mediante auto‑juego sin heurísticas.

La primera versión completa del programa se terminó el 16 de enero de 2026. La siguiente fase consiste en generar todos los datasets de entrenamiento desde cero y documentar el proceso conforme el proyecto evoluciona.

El motor está escrito 100% en Python como un solo archivo .py, es totalmente bilingüe (español/inglés) y funciona tanto en Linux como en Windows.

BB‑Zero incluye tres modos integrados:
- Modo Juego  
- Modo Generación de Dataset  
- Modo Entrenamiento  

Al finalizar el entrenamiento, BB‑Zero será liberado como un ejecutable para Windows (~10 MB) acompañado de su red neuronal *La_Marioneta* (~15 MB) para que cualquiera pueda jugarlo.
